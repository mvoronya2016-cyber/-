Задание 1. Анализ требований 
1. - Пункт 7 + Пункт 13. цена фиксируется при добавлении, цена автоматически обновляется при изменении в каталоге. Противоречие прямое
   - Пункт 3 + Пункт 4. Два ограничения на объем корзины, не понятно какой приоритетнее, могут конфликтовать
   - Пункт 2 + Пункт 9. В П.2 говорится, что для удаления нужна отдельная кнопка, а в П.9 говорится, что юзер может уменьшить кол-во товаров до 0 без отдельной кнопки
2. - В корзине могут находиться разные товары
- Ограничения корзины работают одновременно: не более 5 разных товаров; не более 10 единиц одного товара; общее количество товаров не больше 20
- При добавлении товара, нарушающем любое ограничение, показывается конкретное сообщение с указанием причины
- Пользователь может уменьшать количество товара до 0, то товар автоматически удаляется
- Отдельная кнопка удаления товара должна быть добавлена как альтернативный способ 
- Цена товара фиксируется на момент добавления в корзину и не меняется до оформления заказа
- Автоматическое обновление цены при изменении цены в каталоге не производится
- При возврате пользователя на страницу корзины после изменения цены в каталоге может быть показано информационное сообщение: «Цена на товар X изменилась в каталоге. Если хотите новую цену, удалите и добавьте товар заново»
- В корзине под списком товаров может отображаться рекламный блок с товарами-рекомендациями
- Реклама показывается только в будние дни (пн–пт) по московскому времени: утренний показ с 9:00 до 10:00, вечерний - с 19:00 до 20:00
- Рекламные товары не входят в лимиты корзины и не считаются добавленными в корзину
3. - Какой лимит приоритетнее? По количеству разных товаров, по количеству одного товара или по общему числу штук?
- Откуда взялись данные цифры? Можно их изменить?
- Удаление должно быть только кнопкой или уменьшение количества до 0 тоже удаляет товар?
- Если цена товара в каталоге выросла, а у пользователя в корзине старая цена,то как быть?
- Если цена упала, то показывать новую цену или оставить старую?
- Если товар стал недоступен, то удалять его из корзины автоматически или показывать ошибку при оформлении?\
